## Problem 4 (30pts)

    Associative Lookup = A = 160 nanoseconds
    Hit Ratio = H = .8

    Effective Access Time = (0 + A) * H + (0 + 2 * A)(1 - H)
                          = (0 + 160) * .8 + (0 + 160 * 2)(1 - .8)
                          = 192 nanoseconds
